320 building instructions

Obtain all parts for the product you are assembling. Click "component list" to see which components go with the product you are building

320 Components

ASSEMBLY

Chassis

Remove EN-CH-100 Shapepoint chassis from cardboard box.
Store cardboard box and packing materials for later use.
Remove (4) lid screws.
Throw away silica.
Fasten power supply (EN-PS-101) using 4 countersunk screws
Remove stock expansion card backplate (EN-BP-003) and replace with solid backplate (EN-BP-005) Unscrew stock motherboard back plate (EN-BP-009) and replace with (EN-BP-008).
Store stock back plate for future use.

Install Motherboard

Open (EN-MB-107) J1900 motherboard box.
Install Motherboard using (4) motherboard screws. Put screws in loosely, then tighten.

Install Memory

Insert in slot closest to CPU. Make sure the tabs click in.
Connect Cables to motherboard.
Connect ATX cable
Connect ATX 12v cable
Connect Front Panel connector, insert with the tab pointing towards the power supply
Connect CPU fan.
Connect Sys fan.
Make sure all cables are clear from fans

Install Hard Drive

Unwrap the hard drive
Connect the Hard Drive data cable that came with the motherboard to the drive and
motherboard. Use the straight cable rather than the 90 deg one. Connect it to the SATA 3
port on the motherboard.
Connect the Hard Drive Power cable to the drive.
Install the Hard drive to the enclosure using 4 (EN-SC-001) Chassis screws.

BURN IN

  1. Once software build is complete you will get an Eagle Eye Build confirmation with Bridge Model & Eagle Eye serial #.
  2. Print out Eagle Eye Serial # & attach to buttom of Eagle Eye Bridge.
  3. Press enter & bridge will shutdown Automatically.
  4. Pull USB Software build key from bridge.
  5. Next power back on, you should see Eagle Eye Bridge Model & digit serial #.
  6. Next type in Burnin
  7. Next type in burnintest
  8. Next hit enter.
  9. This will start Burnin Process.
  10. Remove Ethernet, VGA, Keyboard and leave running for days.
  11. Once days are complete then Bridge can now go through function test.
