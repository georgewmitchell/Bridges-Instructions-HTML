320 Components

ASSEMBLY

Chassis Preparation (1U 10")

  1. Remove chassis from cardboard box
  2. Store cardboard box and packing materials for later use.
  3. Remove all lid screws and set aside for later
  4. Attach power supply to chassis with 4 chassis screws
  5. Install expansion card backplate with 2 chassis screws
  6. Install motherboard backplate with 2 chassis screws
  7. If chassis comes with backplates that are not used for this product, put those components back into inventory
  8. Apply CAM/LAN sticker above the 2 ethernet ports on the back of the chassis
  9. Apply UL sticker to the back of the chassis below expansion port. If the unit is a 303/403, apply this sticker to the left side of the chassis closest to the back

Install motherboard (J1900)

  1. Open motherboard box
  2. Insert memory in slots closest to CPU. Make sure the tabs click in
  3. Install Motherboard into chassis using (4) motherboard screws
  4. Put all screws in loosely, then tighten once they are all in

Install Hard Drive

  1. Unwrap the hard drive
  2. Connect the Hard Drive data cable that came with the motherboard to the drive and motherboard. Use the straight cable if your hard drive is 2.5". Use the red left angle cable if your drive is 3.5". Connect the cable to the SATA 0 port on the motherboard.
  3. Connect the Hard Drive Power cable from the power supply to the hard drive.
  4. Install the Hard drive to the enclosure using 4 hard drive screws.

Connect Cables to motherboard.

  1. Connect ATX cable
  2. Connect ATX 12v cable
  3. Connect Front Panel connector, insert with the tab pointing towards the power supply
  4. Connect CPU fan.
  5. Connect Sys fan.
  6. Make sure all cables are clear from fans
  7. Attatch lid with 4 chassis screws put aside earlier

Final assembly steps

  1. Attach lid with the 4 chassis screws
  2. Apply bridge model number sticker

BURN IN

  1. Once software build is complete you will get an Eagle Eye Build confirmation with Bridge Model & Eagle Eye serial #.
  2. Print out Eagle Eye Serial # & attach to buttom of Eagle Eye Bridge.
  3. Press enter & bridge will shutdown Automatically.
  4. Pull USB Software build key from bridge.
  5. Next power back on, you should see Eagle Eye Bridge Model & digit serial #.
  6. Next type in Burnin
  7. Next type in burnintest
  8. Next hit enter.
  9. This will start Burnin Process.
  10. Remove Ethernet, VGA, Keyboard and leave running for days.
  11. Once days are complete then Bridge can now go through function test.
