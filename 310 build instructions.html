310 building instructions

Obtain all parts for the product you are assembling. Click "component list" to see which components go with the product you are building

310 Components

ASSEMBLY

  1. Install system fan with 2 screws onto chassis EN-CH-001 from underneath chassis
  2. Install power supply (EN-PS-101) with 4 screws (2 in back, 2 underneath)
  3. Install zotac backplate (EN-BP-009) 2 screws
  4. Install 8 channel DVR card backplate (EN-DV-102) with 2 screws on to back of chassis
  5. Install 750GB hard drive (EN-DC-100)) with 4 screws( underneath)
  6. Attach Sata cable to hard drive
  7. Install zotac motherboard (EN-MB-106) with 2 screws(back)
  8. Install G1840 processor (EN-CP-108)with triangle matching to motherboard clip on
  9. Install CPU fan (EN-BL-100) push 4 tabs until they click. install fan power cable on motherboard
  10. Install ATX black yellow cable power cable for analog camera a(EN-CB-001) with 2 rounds screws
  11. Attach 8 channel DVR card (EN-DV-102) to PCIE connector (EN-RC-103) Zip tie together(NEW PROCESS)
  12. Install 8 channel DVR card (EN-DV-102) and PCIE connector (EN-RC-103) onto motherboard push down onto motherboard screw tight with 2 round hex screws
  13. Connect power supply(24pin connector) cable to motherboard
  14. Connect system fan cable to motherboard
  15. Connect Sata power supply cable to hard drive
  16. Connect black yellow ATX cable (EN-CB-001) to power supply
  17. Use zip ties to tie off any loose cables or wires(follow cabling routing instructions)
  18. Install lid with 4 screws
  19. Attach 310 sticker (EN-ST-001) to front panel
  20. Attach wan / cam lam sticker (EN-ST-102) to back of unit

BURN IN

  1. Once software build is complete you will get an Eagle Eye Build confirmation with Bridge Model & Eagle Eye serial #.
  2. Print out Eagle Eye Serial # & attach to buttom of Eagle Eye Bridge.
  3. Press enter & bridge will shutdown Automatically.
  4. Pull USB Software build key from bridge.
  5. Next power back on, you should see Eagle Eye Bridge Model & digit serial #.
  6. Next type in Burnin
  7. Next type in burnintest
  8. Next hit enter.
  9. This will start Burnin Process.
  10. Remove Ethernet, VGA, Keyboard and leave running for days.
  11. Once days are complete then Bridge can now go through function test.
