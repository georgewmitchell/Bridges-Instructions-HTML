301 building instructions

Obtain all parts for the product you are assembling. Click "component list" to see which components go with the product you are building

301 Components

ASSEMBLY

Chassis

  1. Remove EN-CH-100 Shapepoint chassis from cardboard box
  2. Store cardboard box and packing materials for later use.
  3. Remove (4) lid screws.
  4. Throw away silica.
  5. Fasten power supply (EN-PS-101) using 4 countersunk screws
  6. Remove stock expansion card backplate (EN-BP-003) and replace with solid backplate (EN-BP-005)
  7. Unscrew stock motherboard back plate (EN-BP-009) and replace with (EN-BP-008).
  8. Store stock back plate for future use.

Install Motherboard

  1. Open (EN-MB-107) J1900 motherboard box.
  2. Install Motherboard using (4) motherboard screws.
  3. Put screws in loosely, then tighten.

Install Memory

  1. Insert in slot closest to CPU. Make sure the tabs click in.
  2. Connect Cables to motherboard.
  3. Connect ATX cable
  4. Connect ATX 12v cable
  5. Connect Front Panel connector, insert with the tab pointing towards the power supply
  6. Connect CPU fan.
  7. Connect Sys fan.
  8. Make sure all cables are clear from fans

Install Hard Drive

  1. Unwrap the hard drive
  2. Connect the Hard Drive data cable that came with the motherboard to the drive and
  3. motherboard. Use the straight cable rather than the 90 deg one. Connect it to the SATA 3
  4. port on the motherboard.
  5. Connect the Hard Drive Power cable to the drive.
  6. Install the Hard drive to the enclosure using 4 hard drive screws.

BURN IN

  1. Once software build is complete you will get an Eagle Eye Build confirmation with Bridge Model & Eagle Eye serial #.
  2. Print out Eagle Eye Serial # & attach to buttom of Eagle Eye Bridge.
  3. Press enter & bridge will shutdown Automatically.
  4. Pull USB Software build key from bridge.
  5. Next power back on, you should see Eagle Eye Bridge Model & digit serial #.
  6. Next type in Burnin
  7. Next type in burnintest
  8. Next hit enter.
  9. This will start Burnin Process.
  10. Remove Ethernet, VGA, Keyboard and leave running for days.
  11. Once days are complete then Bridge can now go through function test.
