Archiver 03 Components

ASSEMBLY

Chassis

Remove chassis from cardboard box.
Store cardboard box and packing materials for later use.
Remove (4) lid screws.
Remove hard drive cages from chassis and put aside for later
Fasten power supply using 4 chassis screws

Install Motherboard

Open motherboard box.
Install Motherboard using (4) motherboard screws
Insert the 2 SATA cards into the motherboards. Fasten them to the chassis using the expansion port screws

Install Memory

Insert memory into slots on motherboard. Make sure the tabs click in.

Install Hard Drives

Unwrap 12 hard drive
Install hard drives into cages using 4 chassis screws per drive. Each chassis will get 4 hard drive assemblies
Install the 4 hard drive assemblies into the chassis
Insert the drive assembly so that the lip pushes into the drive assembly slot
Each assembly will use a singe chassis screw to secure to the chassis.
Install SSD drive onto system drive mount
Attach that system drive assembly to the chassis

Connect Cables to motherboard.

Connect 24 pin ATX cable
Connect ATX 12v cable (black and yellow wire)
Connect Front Panel connector, insert with the tab pointing towards the power supply
Connect CPU fan to motherboard
Connect chassis fans to power supply.
Make sure all cables are clear from fans
Connect hard drive power cables to the 12 drives
Connect hard drive power cable to the SSD system drive
Connect Sata cables to all of the drives. The system drive should be plugged into SATA port 0
