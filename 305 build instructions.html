305 building instructions

Obtain all parts for the product you are assembling. Click "component list" to see which components go with the product you are building

305 Components

ASSEMBLY

  1. Unscrew bottom plate.
  2. Use metal hook tool to pull plate off of chassis
  3. Unscrew PC board and detach from chassis
  4. Insert Memory into Memory slot. Push in and Push Down.
  5. Screw PC board back into place with screws. Use Phillips screwdriver to prevent stripping.
  6. Remove blue plastic from bottom plate.
  7. Install hard drive onto bottom plate with supplied screws.
  8. Place bottom plate back onto chassis and screw down with screws.

BURN IN

  1. Once software build is complete you will get an Eagle Eye Build confirmation with Bridge Model & Eagle Eye serial #.
  2. Print out Eagle Eye Serial # & attach to buttom of Eagle Eye Bridge.
  3. Press enter & bridge will shutdown Automatically.
  4. Pull USB Software build key from bridge.
  5. Next power back on, you should see Eagle Eye Bridge Model & digit serial #.
  6. Next type in Burnin
  7. Next type in burnintest
  8. Next hit enter.
  9. This will start Burnin Process.
  10. Remove Ethernet, VGA, Keyboard and leave running for days.
  11. Once days are complete then Bridge can now go through function test.
