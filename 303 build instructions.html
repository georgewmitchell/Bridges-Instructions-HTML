303 Components

ASSEMBLY

  • 1.Install system fan with two screws onto chassis EN-CH-001
  • 2.Install hard drive 750 GB(EN-DC-100)
  • 3.Install J1900 motherboard backplate(EN-BP-008)
  • 4.Install POE ethernet board backplate(EN-BP-006)
  • 5.Install 9 Port POE switch board(EN-CA-102)
  • 6.Install J1900 motherboard(EN-MB-107) with two screw
  • 7.Install 4 GB RAM(EN-ME-105) onto motherboard
  • 8.Install power supply with black PowerBrick (EN-PS-105) (solder purple/black wire to power supply)
  • 9. Install PCI ethernet blue cable(EN-CA-100) to POE board
  • 10.Install PCI ethernet card to motherboard
  • 11.Install 750 GB(EN-DC-100) hard drive data cable to motherboard open data port
  • 12.Install system fan to motherboard
  • 13.Install power supply ATX cable to motherboard
  • 14.Install EN-PS-106 Power Supply

    BURN IN

    1. Once software build is complete you will get an Eagle Eye Build confirmation with Bridge Model & Eagle Eye serial #.
    2. Print out Eagle Eye Serial # & attach to buttom of Eagle Eye Bridge.
    3. Press enter & bridge will shutdown Automatically.
    4. Pull USB Software build key from bridge.
    5. Next power back on, you should see Eagle Eye Bridge Model & digit serial #.
    6. Next type in Burnin
    7. Next type in burnintest
    8. Next hit enter.
    9. This will start Burnin Process.
    10. Remove Ethernet, VGA, Keyboard and leave running for days.
    11. Once days are complete then Bridge can now go through function test.
