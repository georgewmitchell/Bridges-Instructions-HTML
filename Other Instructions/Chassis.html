Chassis Preparation (1U 10")

  1. Remove chassis from cardboard box
  2. Store cardboard box and packing materials for later use.
  3. Remove all lid screws and set aside for later
  4. Attach power supply to chassis with 4 chassis screws
  5. Install expansion card backplate with 2 chassis screws
  6. Install motherboard backplate with 2 chassis screws
  7. If chassis comes with backplates that are not used for this product, put those components back into inventory
  8. Apply CAM/LAN sticker above the 2 ethernet ports on the back of the chassis
  9. Apply UL sticker to the back of the chassis below expansion port. If the unit is a 303/403, apply this sticker to the left side of the chassis closest to the back

Chassis Preparation (Barebones 304)

  1. Remove barebones chassis from box
  2. Unscrew rubber pegs from bottom of chassis and remove bottom lid
  3. Unscrew metal hard drive bracket from inside chassis
  4. Attach hard drive to long side of metal bracket
  5. Insert memory into motherboard memory slot. Line up slots and snap in
  6. Insert hard drive into port on motherboard and fasten assembly with provided screws onto chassis
  7. Place lid and fasten rubber button pegs
  8. Apply model number sticker to front of bridge

Chassis Preparation (Barebones 305)

  1. Unscrew bottom plate.
  2. Use metal hook tool to pull plate off of chassis
  3. Unscrew PC board and detach from chassis
  4. Insert Memory into Memory slot. Push in and Push Down.
  5. Screw PC board back into place with screws. Use Phillips screwdriver to prevent stripping.
  6. Remove blue plastic from bottom plate.
  7. Install hard drive onto bottom plate with supplied screws.
  8. Place bottom plate back onto chassis and screw down with screws.
