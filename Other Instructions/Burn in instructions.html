Instructions for burn in

  1. Once software build is complete you will get an Eagle Eye Build confirmation with Bridge Model & Eagle Eye serial #.
  2. Print out Eagle Eye Serial # & attach to buttom of Eagle Eye Bridge.
  3. Press enter & bridge will shutdown Automatically.
  4. Pull USB Software build key from bridge.
  5. Next power back on, you should see Eagle Eye Bridge Model & digit serial #.
  6. Next type in Burnin
  7. Next type in burnintest
  8. Next hit enter.
  9. This will start Burnin Process.
  10. Remove Ethernet, VGA, Keyboard and leave running for days.
  11. Once days are complete then Bridge can now go through function test.
