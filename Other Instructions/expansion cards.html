Install DVR Card

  1. Install expansion port backplate to chassis using 2 chassis screws
  2. Insert DVR card into riser card PCIE connector
  3. Zip tie this assembly together !!!(NEW PROCESS)!!!
  4. Fasten analog power cables to the expansion card backplate
  5. Insert the assembly into PCIE slot on motherboard
  6. Fasten assembly to the expansion card backplate using the 2 supplied round hex screws that came with the card

Install POE Switchboard

  1. Install expansion port backplate to chassis using 2 chassis screws
  2. Attatch 9 Port POE switch board to chassis using 3 motherboard screws
  3. Install J1900 motherboard(EN-MB-107) with two screw
  4. Install 4 GB RAM(EN-ME-105) onto motherboard
  5. Install power supply with black PowerBrick (EN-PS-105) (solder purple/black wire to power supply)
  6. Install PCI ethernet blue cable(EN-CA-100) to POE board
  7. Install PCI ethernet card to motherboard
  8. Install 750 GB(EN-DC-100) hard drive data cable to motherboard open data port
  9. Install system fan to motherboard
  10. Install power supply ATX cable to motherboard
  11. Install EN-PS-106 Power Supply
