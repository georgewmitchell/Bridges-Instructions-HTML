Install motherboard (J1900)

  1. Open motherboard box
  2. Install Motherboard into chassis using (4) motherboard screws
  3. Put all screws in loosely, then tighten once they are all in

Install motherboard (Zotac H81)

  1. Open motherboard and CPU boxes and take out the components
  2. Install CPU. Insert CPU into socket with the triangle aligning with the triangle on the CPU socket. Secure the CPU by clipping and closing the socket latch
  3. Install CPU blower. Place the blower over CPU and align this with the holes in the motherboard. The blower should be directed where the airflow is toward the rear of the chassis
  4. Press the 4 tabs from the blower into the motherboard socket holes until they click
  5. Plug in blower power cable into motherboard where it's labled CPU fan
  6. Install Motherboard into chassis using (4) motherboard screws
  7. Put all screws in loosely, then tighten once they are all in
