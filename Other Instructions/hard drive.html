Install Hard Drive

  1. Unwrap the hard drive
  2. Connect the Hard Drive data cable that came with the motherboard to the drive and motherboard. Use the straight cable if your hard drive is 2.5". Use the red left angle cable if your drive is 3.5". Connect the cable to the SATA 0 port on the motherboard.
  3. Connect the Hard Drive Power cable from the power supply to the hard drive.
  4. Install the Hard drive to the enclosure using 4 hard drive screws.
