Obtain all parts for the product you are assembling. Click "component list" to see which components go with the product you are building

Server 03 Components

ASSEMBLY

Chassis

Remove chassis from cardboard box.
Store cardboard box and packing materials for later use.
Remove (4) lid screws.
Fasten power supply using 4 chassis screws

Install Motherboard

Open motherboard box.
Install Motherboard using (4) motherboard screws

Install Memory

Insert 4 memory sticks into slots on motherboard. Make sure the tabs click in.

Install Hard Drives

Unwrap 2 SSD hard drive
Install the 2 hard drive assemblies into the chassis
Install SSD drives in the front of the chassis using 4 hard drive screws per drive
Attach that system drive assembly to the chassis

Connect Cables to motherboard.

Connect 24 pin ATX cable
Connect ATX 12v cable (black and yellow wire)
Connect Front Panel connector to the motherboard
Connect CPU fan to motherboard
Connect chassis fans to motherboard.
Make sure all cables are clear from fans
Connect hard drive power cable to the 2 SSD drives
Connect Sata cables to all of the drives.
